## Getting Started

1. Install dependencies:
```bash
pnpm install
```

2. Run the development server:
```bash
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

- Update the content in each component (`Hero.tsx`, `About.tsx`, `Portfolio.tsx`, `Contact.tsx`)
- Replace placeholder images with actual project images
- Modify colors and styling in `globals.css` and `tailwind.config.ts`
- Update contact email in `Contact.tsx`

## Build for Production

```bash
pnpm build
pnpm start
```

## Tech Stack

- **Next.js 14** - React framework
